Essa experiência internacional me proporcionou habilidades valiosas como resolução de problemas, resiliência e comunicação, que agora aplico no meu objetivo de me tornar um Desenvolvedor Backend completo.

- 🎓 Cursando **Engenharia de Software** para solidificar minha base teórica.
- 📚 Atualmente focado no ecossistema **Python**, aprofundando meus conhecimentos em **FastAPI, SQL, SOLID e Design Patterns**.
- 💼 Buscando ativamente por uma **oportunidade de estágio** onde eu possa contribuir, aprender e crescer profissionalmente.

### 2. Sistema Gerador de Orçamentos
<p>Ferramenta de linha de comando (CLI) desenvolvida em Python para otimizar a criação de propostas comerciais. A aplicação utiliza SQLAlchemy como ORM para gerenciar clientes e orçamentos em um banco de dados SQLite, gera propostas profissionais em PDF e tem sua qualidade de código garantida por testes unitários com Pytest.</p>
